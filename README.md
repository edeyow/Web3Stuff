# Web3Stuff

Various scripts and tools for Web3 projects — whitelists, minters, recon tools, and more.

## Structure

- **`Live/`** — Active, usable scripts for ongoing projects
- **`Outdated/`** — Scripts for completed, launched, or dead projects

## Usage

Most scripts are paste-in-console tools designed to run directly in a browser's DevTools console on the target site. Some are Node.js scripts meant to run server-side.

Check each project's folder for specific instructions.
